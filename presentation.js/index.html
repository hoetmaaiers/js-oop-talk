Constructor

  • Creates different instances aka Objects
  • Use new operator
  • Accepts parameters
  • Returns an instance aka Object

              function Car(name){
                name: name,
                toot: function() {
                  console.log("Tuut Tuuuuuuuut");
                }
              }
            

              var bug = new Car("bug");

Prototype

  • A property
  • Defines properties and methods to all instances
  • Works with new operator
  • Stored int he memory once

              function Car(name){
                this.name = name;
                this.wheels = 4;
              }
            

              Car.prototype.view = function() {
                return this.name + " has " + this.wheels + ".";
              }

              var bug = new Car("bug");
              bug.view(); // "Bug has 4 wheels."
            

              Car.prototype.drive = function() {
                return this.name + " is cruising";
              }

              bug.drive(); // "Bug is cruising"

Inheritance

Inheritance

  • A new "class" is created from another "class"
  • Extends functionality
  • Defines new attributes and methods
  • A way to reuse code
  • Object inherits from Object.prototype
  • multiple ways to inherit in JS

One way to solve inheritance in JS


              function Car(name){
                this.name = name,
                this.wheels = 4
              }

              Car.prototype.drive = function() {
                return this.name + " is cruising";
              }
            


              function SportCar = function() {
                this.name = "name",
                this.wheels = 4
              }

              SportCar.prototype = new Car()

              SportCar.prototype.driveFast = function() {
                return this.name + " is driving faster then the speed of light!"
              }
            

              var tesla = new SportCar("Tesla");
              
              tesla.drive(); // "Tesla is cruising."
              tesla.driveFast() // is driving faster then the speed of light!
